#  Приложение Unsplash Photos
## Тестовое задание
Приложение работающее с REST API сервиса [Unsplash](https://unsplash.com).

**Язык**: Swift 5.

**Поддерживаемая iOS**: 13.

**Фреймворк**: UIKit, верстка кодом без Storyboard.

**Используемые библиотеки**: Kingfisher, Realm.
